# Responsive Survey Form

Formulário de pesquisa responsivo desenvolvido com HTML e CSS como prática de desenvolvimento front-end.

---

## 📌 Objetivo

Este projeto foi criado para praticar a criação de formulários, organização de layout e estilização utilizando HTML e CSS.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

---

---

## 📂 Como executar o projeto

1. Clone este repositório
2. Abra o arquivo `index.html` no navegador

---

## 📚 Aprendizados

Durante este projeto pratiquei:

- Estruturação de formulários em HTML
- Uso de labels e inputs
- Campos de seleção (select)
- Radio buttons e checkboxes
- Organização de layout com CSS
- Boas práticas de estrutura HTML
